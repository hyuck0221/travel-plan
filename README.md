# ✈️ 여행일정 관리

**https://travel.hspace.site**

설치도, 로그인도 필요 없는 여행 일정 웹 서비스입니다.
일정을 작성하면 URL에 자동으로 저장되고, 그 링크를 공유하면 상대방이 바로 확인할 수 있습니다.

---

## 시작하기

브라우저에서 **https://travel.hspace.site** 에 접속하면 바로 사용할 수 있습니다.
회원가입, 로그인, 앱 설치 모두 필요하지 않습니다.

---

## 사용 방법

### 일정 추가

일정을 추가하는 방법은 세 가지입니다.

**① 직접 입력**
좌측 패널 하단의 `+ 일정 추가` 버튼을 클릭하면 새 카드가 생성됩니다.
장소명·주소·날짜·시간·메모를 직접 입력할 수 있습니다.

**② 지도 검색**
지도 상단 검색창에 장소명을 입력하면 결과 목록이 나타납니다.
원하는 장소를 클릭하면 지도에 핀이 표시되고, `일정 등록` 버튼을 눌러 일정에 추가합니다.

**③ 지도 클릭**
지도에서 원하는 위치를 클릭하면 해당 위치에 핀이 꽂히고 주소가 자동으로 입력됩니다.
`일정 등록` 버튼을 눌러 일정에 추가합니다.

---

### 일정 편집

카드를 **더블클릭**하거나 우측 상단 편집 버튼을 클릭하면 편집 모드로 전환됩니다.

| 필드 | 설명 |
|------|------|
| 장소 | 장소명 입력 — 입력 중 자동 검색 결과 표시 |
| 주소 | 주소 입력 — 입력 중 자동 검색 결과 표시 |
| 날짜 | `변경` 버튼 클릭 → 캘린더에서 선택 |
| 시간 | `변경` 버튼 클릭 → 시간 선택기에서 설정 |
| 메모 | 자유롭게 텍스트 입력 |

> 새 일정을 추가하면 바로 직전 일정의 날짜와 시간이 자동으로 입력됩니다.

---

### 일정 순서와 정렬

- 날짜가 설정된 일정은 날짜 → 시간 순으로 자동 정렬됩니다.
- 날짜가 없는 일정은 목록 하단에 표시됩니다.
- 시간을 변경하면 즉시 순서가 바뀝니다.
- 지도의 번호 핀도 이 순서에 맞춰 자동 갱신됩니다.

---

### 지도 활용

**핀 번호**
날짜·시간 순으로 1번부터 번호가 부여됩니다. 선택된 일정의 핀은 주황색으로 강조됩니다.

**경로선**
핀이 2개 이상이면 방문 순서대로 선으로 연결되어 동선을 한눈에 파악할 수 있습니다.

**현위치**
지도 우하단 버튼을 클릭하면 현재 위치가 빨간 점으로 표시되고 실시간으로 업데이트됩니다.
다시 클릭하면 끕니다.

**패널 크기 조절**
좌측 일정 패널과 지도 사이의 구분선을 드래그해서 원하는 비율로 조절할 수 있습니다.

---

### 공유하기

**링크 공유**
상단 `공유` 버튼을 클릭하면 현재 일정이 담긴 URL이 공유됩니다.
모바일에서는 기기 공유 기능이 열리고, 데스크톱에서는 클립보드에 복사됩니다.

**단축 링크 복사**
`링크 단축하여 복사` 버튼을 클릭하면 긴 URL을 짧게 줄여 클립보드에 복사합니다.
같은 URL은 자동으로 캐싱되어 반복 클릭 시 즉시 복사됩니다.

**QR 코드**
`QR` 버튼을 클릭하면 현재 일정의 QR 코드가 생성됩니다.
이미지를 저장해 오프라인에서도 공유할 수 있습니다.

> 공유받은 링크를 열면 해당 일정이 내 일정 목록에 자동으로 추가됩니다.

---

### 여러 일정 관리

상단 가운데 드롭다운에서 여러 여행 일정을 전환할 수 있습니다.

- `새 일정 만들기` — 새 여행 일정을 추가합니다.
- 일정 이름 클릭 — 해당 일정으로 전환합니다.
- × 버튼 — 해당 일정을 삭제합니다.

모든 일정은 기기의 로컬 스토리지에도 자동 저장되어 브라우저를 닫아도 유지됩니다.

---

### 실행 취소 / 다시 실행

| 동작 | 단축키 |
|------|--------|
| 실행 취소 | `Ctrl + Z` (Mac: `⌘ + Z`) |
| 다시 실행 | `Ctrl + Shift + Z` (Mac: `⌘ + Shift + Z`) |

상단 ← → 버튼으로도 동일하게 사용할 수 있습니다.

---

## 데이터 저장 방식

별도 서버에 데이터를 저장하지 않습니다.

- **URL 해시** — 현재 활성 일정의 전체 내용이 URL에 인코딩됩니다. URL만 있으면 언제 어디서든 동일한 일정을 볼 수 있습니다.
- **로컬 스토리지** — 모든 일정이 사용 중인 기기에 자동 저장됩니다. 브라우저를 닫아도 데이터가 유지됩니다.

> 브라우저 데이터를 삭제하면 로컬 저장 데이터도 함께 삭제됩니다. 중요한 일정은 URL을 따로 보관해두세요.
